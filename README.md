# how-tos
一、如何注册Yandex邮箱
1、打开网址：https://mail.yandex.com （会自动跳转至 https://360.yandex.com/mail/ ）
2、点击页面 右上角的 Log in
3、在跳转过来的页面上，在“Username or email”文本框中，输入“abcde@efghijk.lmn” （你现有的email地址）
